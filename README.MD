# 介绍

本项目为Vue3开发的PDF + MP3播放器，主要使用场景为方便PDF和MP3组合阅读，例如：Pdf + 音频的网课，教程等等；

使用IndexDB存储完整的文件以及音频对象。完全本地化，无服务端。

## 功能


### v1.0.0

1.打开本地pdf文档
2.为文档上传对应的音频mp3文件
3.列表切换文件
4.支持pdf最小0x最大2x缩放
5.自定义跳转页码
